# UI for Colorbot

Why? If you have a ColorBot for antich3ats such as like Vanguard, its risky to have UI on ImGUI because it will take a screenshot and ur cooked.
If u have a LocalHost UI or even Stealthier on Phone, there is nothing to screenshot.

Add POST, buttons, sliders, connection and ur free to go.
## Features:
* Remote access from devices on the LAN (Wi-Fi).
* Entire Public folders are embedded into the EXE.

# How to run?
* Install Rust and VS (with C++ reds to even build this)
* go to the folder and open cmd and type 'cargo build --release'
* Builded exe will be in /target/release
